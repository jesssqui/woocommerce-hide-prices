Professional plugin that hides prices and purchase options from unauthorized users. Perfect for wholesale, B2B, and restricted access stores.

== Description ==

**WooCommerce Hide Prices** is a comprehensive solution for creating restricted access WooCommerce stores. Hide prices and purchase options from unauthorized users while providing a professional experience for approved customers.

= Key Features =

* **Global Price Hiding**: Hide all prices with one click
* **Custom User Roles**: Built-in wholesale_customer and vip_customer roles
* **Role-Based Access**: Flexible user role management
* **Request Access System**: Configurable access request functionality
* **Product-Specific Controls**: Override global settings per product
* **Category-Based Restrictions**: Apply restrictions to entire categories
* **Custom Styling**: Add your own CSS for perfect integration
* **User Expiration**: Set access expiration dates for users
* **Professional Admin Interface**: Easy-to-use settings panel

= Perfect For =

* **Wholesale Stores**: Restrict pricing to approved wholesale customers
* **B2B Websites**: Create member-only purchasing experiences
* **Private Stores**: Limit access to exclusive customer groups
* **Membership Sites**: Integrate with existing membership systems

= Advanced Features =

* **Configurable Request Access URL**: Direct users to your contact form or application page
* **Hierarchical Settings**: Product → Category → Global settings priority
* **High-Performance Order Storage (HPOS) Compatible**
* **Translation Ready**: Full internationalization support
* **Clean Uninstall**: Removes all data when deleted

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/woocommerce-hide-prices/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Settings → Hide Prices to configure the plugin
4. Assign users to wholesale_customer or vip_customer roles
5. Configure your request access settings

== Frequently Asked Questions ==

= Does this work with all WooCommerce themes? =

Yes, the plugin is designed to work with any properly coded WooCommerce theme.

= Can I customize the "Request Access" button? =

Yes, you can configure the button text, message, and destination URL in the settings.

= Are the custom user roles automatically created? =

Yes, wholesale_customer and vip_customer roles are created automatically when the plugin is activated.

= Can I hide prices for specific products only? =

Yes, you can override global settings on a per-product basis using the product meta boxes.

= Is this compatible with WooCommerce HPOS? =

Yes, the plugin is fully compatible with High-Performance Order Storage.
